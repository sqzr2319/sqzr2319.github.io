电工电子技术基础

大二下学期电工电子技术基础的复习笔记，目前更新至模拟电路。

一、电路原理

1. 直流电路

  1. 电源等效变换
    1. 开路是电流为零的恒流源，短路是电压为零的恒压源
    2. 恒压源并电阻等于没并，恒流源串电阻等于没串
    3. 恒压源串电阻可等效为恒流源并电阻
    4. 受控源也可进行
  2. 广义 KCL：三端网络 $i_1+i_2+i_3=0$，二端网络 $i_i=i_o$，一端网络 $i=0$
  3. 支路电流法：$N$ 结点 $B$ 支路，设支路电流，列 $N-1$ 个独立 KCL 和 $B-N+1$ 个独立 KVL
  4. 节点电位法：取参考点，设未知结点电位，列 $N-1$ 个独立 KCL
    1. 左边：未知结点电位×Σ与该结点相连的不含恒流源支路电导-相邻结点电位×Σ与未知结点共有支路电导
    2. 右边：Σ与该结点相连的含恒压源支路电动势×本支路电导（电动势朝向该结点时为正）+Σ与该结点相连的含恒流源支路电流（电流朝向该结点时为正）
    3. 带受控源电路列方程时增加受控源关系式
  5. 线性电路叠加原理、齐性定理：电压电流满足功率不满足；受控源不能单独作用于电路
  6. 替代定理：已知某支路电压或电流，将该支路替换为恒压源或恒流源
  7. 戴维南等效：等效电压源电压等于有源二端网络的开路电压，内阻等于有源二端网络相应无源二端网络的等效电阻
  8. Y-Δ 转换：$Y_A=\frac{Δ_{AB}Δ_{AC}}{Δ_{AB}+Δ_{BC}+Δ_{AC}}$，$Δ_{AB}=Y_A+Y_B+\frac{Y_AY_B}{Y_C}$（相等时 $Y=Δ/3$）
  9. 平衡电桥：$R_1R_4=R_2R_3$，$R_5$ 可取任意值（短路或开路）
  10. 负载电阻等于电源内阻时，负载功率最大
  11. 二端口网络：输入电阻去除独立源保留受控源后加压求流，输出电阻需短路输入端

2. 正弦交流电路

  1. $\Delta\varphi=\varphi_1-\varphi_2\gt 0$ 称领先，反之称落后
  2. 瞬时值 $u$，最大值 $U_m$，有效值 $U$，相量 $\dot{U}$，复数 $U\angle\varphi$
  3. 电感：$u=L\frac{di}{dt}$，直流中相当于短路，交流复数感抗 $X_L=j\omega L$，无功功率 $Q=UI$ 表示瞬时功率最大值，单位 var
  4. 电容：$i=C\frac{du}{dt}$，直流中相当于开路，交流复数容抗 $X_C=-j\frac{1}{\omega C}$，无功功率 $Q=-UI$
  5. 视在功率 $S=UI$，单位 $V\cdot A$，复数阻抗 $Z=|Z|\angle\varphi$
  6. 有功功率 $P=UI\cos(\varphi_u-\varphi_i)$，无功功率 $Q=UI\sin(\varphi_u-\varphi_i)$
  7. 提高功率因数：并联电容 $C=\frac{P}{\omega U^2}(\tan\varphi_L-\tan\varphi_{CL})$
  8. 负载阻抗可任意改变时，$Z_{外}=Z_{内}^*$ 时功率最大；只可改变电抗则 $X_{外}=-X_{内}$
  9. 品质因数：串联/电压谐振电感或电容上的电压与总电压之比
  10. 上下限截止频率：$I=\frac{I_m}{\sqrt{2}}$，通频带带宽 $BW=f_{C2}-f_{C1}=\frac{R}{2\pi L}=\frac{f_0}{Q}$
  11. 并联/电流谐振：RL与C并联，$\omega_0=\sqrt{\frac{1}{LC}-\frac{R^2}{L^2}}$，总阻抗 $Z_0=\frac{L}{RC}$ 接近最大值
  12. 理想串联谐振相当于短路，理想并联谐振相当于开路

3. 一阶暂态分析

  1. 换路定理：RC或RL电路换路瞬间电容电压和电感电流不跃变
  2. 初始值：$u,i$ 在 $t=0^+$ 的值，$u_C,i_L$ 用换路定理确定，其他用电路方程确定
  3. 三要素法：$f(t)=f(\infty)+[f(0^+)-f(\infty)]e^{-\frac{t}{\tau}},\tau=RC$ 或 $\frac{L}{R}$，$R$ 为电容或电感两端有源二端网络等效电阻，$t=5\tau$ 达到稳态值
  4. 多电容电路是否为一阶电路：去除独立源后电容能否等效为一个
  5. 有无穷大电流冲击时：换路定理不成立，使用电荷守恒（两电容电荷的变化量相等）
  6. 同时含电容和电感的电路是否为一阶电路：两元件间是否相互独立（换路后L和C部分电路与恒流源串联或与恒压源并联）

二、模拟电路

4. 半导体器件

  1. 二极管：击穿区、截止区、导通区，反向击穿电压、反向漏电流 μA、死区电压 0.5V、导通压降 0.7V，小信号等效电阻 $r_D=\Delta u_D/\Delta i_D$
  2. 理想二极管：死区电压等于导通压降等于零，反向电阻无穷大
  3. 稳压二极管：击穿区，电流在一定范围内时电压近似为常数
  4. 三极管：截止区 $U_{BE}$ 小于死区电压，饱和区集电结正偏 $U_{CE}=0.3V$，线性放大区发射结正偏集电结反偏

5. 基本放大电路

  1. 直流量 $U_A$，交流量 $u_a$，全量 $u_A$，电压放大倍数 $\dot{A_u}=A_u\angle\varphi$
  2. 三极管小信号等效电路：$r_{be}=300\Omega+(1+\beta)\frac{26\text{ mV}}{I_E\text{ mA}}$
  3. 共射级放大电路：$\dot{A_u}=-\beta\frac{R_C||R_L}{r_{be}}$
  4. 分压偏置式：$V_B\approx\frac{R_2}{R_1+R_2}V_{CC},I_C\approx I_E$
  5. 信号源带内阻：$\dot{A_{us}}=\dot{A_u}\frac{r_i}{r_i+R_S}$
  6. 带交流射极负反馈电阻：$\dot{A_u}=-\frac{\beta(R_C||R_L)}{r_{be}+(\beta+1)R_F},r_i=R_{B1}||R_{B2}||(r_{be}+(\beta+1)R_F)$
  7. 共集电极放大器：$\dot{A_u}=\frac{(1+\beta)(R_E||R_L)}{r_{be}+(1+\beta)(R_E||R_L)},r_i=R_B||(r_{be}+(1+\beta)(R_E||R_L)),r_o=R_E||\frac{r_{be}+(R_S||R_B)}{1+\beta}$
  8. 差分放大电路：共模分量 $u_c=\frac{u_{i1}+u_{i2}}{2}$ 大小相等极性相同，差模分量 $u_d=\frac{u_{i1}-u_{i2}}{2}$ 大小相等极性相反，差模放大倍数 $\dot{A_d}=\frac{u_o}{2u_d}=-\frac{\beta (R_C||\frac{R_L}{2})}{R_B+r_{be}}$，单端输出 $\dot{A_d}=-\frac{\beta (R_C||R_L)}{2(R_B+r_{be})}$

6. 负反馈

  1. 开环放大倍数 $\dot{A_o}=\frac{\dot{X_o}}{\dot{X_d}}$，闭环放大倍数 $\dot{A_f}=\frac{\dot{X_o}}{\dot{X_i}}$，理想负反馈 $\dot{A_f}=\frac{\dot{A_o}}{1+\dot{A_o}\dot{F}}$
  2. 电流/电压：指输出量，短路输出端后仍有反馈为电流
  3. 串联/并联：指反馈量（并联为电流），输入和反馈在相同端为并联
  4. 三极管极性传递：$u_B↑\Rightarrow u_C↓,u_B↑\Rightarrow u_E↑,u_E↑\Rightarrow u_C↑$
  5. 集成运放极性传递：$u_+↑\Rightarrow u_o↑,u_-↑\Rightarrow u_o↓$
  6. 串联反馈，反馈和输入极性相同为负反馈；并联反馈，反馈和输入极性相反为负反馈
  7. 深度负反馈：$\dot{A_F}=\frac{1}{\dot{F}},\dot{X_i}=\dot{X_f},\dot{X_d}=0$
  8. 电压放大倍数和输入输出电阻（$1+A_oF$ 倍）
    1. 电压串联：$\frac{1}{\dot{F_{uu}}}$，输入增大输出减小
    2. 电压并联：$\frac{1}{\dot{F_{iu}}}\cdot\frac{1}{R_S}$，输入输出减小
    3. 电流串联：$\frac{1}{\dot{F_{ui}}}\cdot R_{L^\prime}$，输入输出增大
    4. 电流并联：$\frac{1}{\dot{F_{ii}}}\cdot\frac{R_{L^\prime}}{R_S}$，输入减小输出增大

7. 运放应用

  1. 分析方法：虚短路（$u_+=u_-$）和虚断路（$i_+=i_-=0$）
  2. 平衡电阻：输入端和输出端都接地，计算对地等效电阻
  3. 非线性状态：虚短路不成立，虚断路成立
  4. 方波发生器：$T=2RC\ln(1+2\frac{R_1}{R_2})$
  5. 三角波发生器：$T=4RC\frac{R_1}{R_2}$，锯齿波 $T=2(R+R^\prime)C\frac{R_1}{R_2}$
  6. 自激振荡：正反馈，$\dot{A_u}\dot{F}=1$
  7. 文氏桥（RC串并联选频网络）：$\omega_0=\frac{1}{\sqrt{R_1R_2C_1C_2}}=\frac{1}{RC}$，$\dot{F}=\frac{1}{3+j(\frac{\omega}{\omega_0}-\frac{\omega_0}{\omega})}$
  8. 三点式LC并联电路：中间端交流接地则首尾极性相反；首尾接地则另两端极性相同
